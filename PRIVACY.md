# Privacy Policy

## Text Match Slicer

Text Match Slicer is a Power BI custom visual published by Whippet Dev.

This privacy policy explains how the visual handles data when used in Power BI reports.

## Data handling

Text Match Slicer runs inside Power BI and uses the data supplied to it by Power BI through the visual data roles.

The visual uses the bound text column and the user's search terms to evaluate matching values and apply a Power BI filter back to the report.

## External services

Text Match Slicer does not intentionally:

* send report data to external services
* make external network requests
* collect telemetry
* use cookies
* use browser local storage
* store user search terms outside the Power BI report session
* require special Power BI visual privileges

## Report data

The visual can process text values that report authors bind to it. The sensitivity of that data depends on the report and dataset chosen by the report author.

Report authors are responsible for ensuring that any fields added to the visual are appropriate for their users and comply with their organisation's data policies.

## Search terms

Search terms entered by users are used only by the visual to calculate matching text values and apply a filter in Power BI.

Search terms are not intentionally sent to Whippet Dev or any external service.
